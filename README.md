# Insightful POS

Point-of-sale web app for restaurant and retail workflows, including:
- sales + orders
- customer credit and statements
- receivables and payment tracking
- PWA install support

## Local development

Requirements:
- Node.js 18+
- npm

Run locally:

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
npm run preview
```

## Stack

- React + TypeScript
- Vite
- Tailwind CSS + shadcn/ui
- Supabase
